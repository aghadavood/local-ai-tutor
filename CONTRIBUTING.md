# Contributing

Got an old or weak laptop? Help map what local AI *actually* runs on normal hardware.

1. Run the benchmark: see [`benchmark/README.md`](benchmark/README.md)
2. Add your finished CSV to `results/` named `results_<yourmachine>.csv`
3. Add a row to `results/LEADERBOARD.md` with your specs
4. Open a pull request

Other languages welcome too — fork the `TASKS` list in `benchmark/benchmark.py`,
swap in your language pair, and share what you find.
